## Summary

| Specification      | Value             |
| :----------------- | :---------------- |
| Aria Role          | `list`            |
| Accessibility name | prop `aria-label` |

:::info{title="Info"}
An alternative text label must be provided to identify the timeline. This should be added using the `aria-label` prop.
:::

#### TimelineItem

- Each item container has the `listitem` role
- Each option is given an accessible name when the prop `date` is assigned
