## Basic usage

<Preview>
  <Demo />
</Preview>

```tsx sandbox
const Demo = () => {
  const [checked, setChecked] = useState(false);

  return (
    <CheckboxField
      label="Include receipt"
      isChecked={checked}
      onChange={(event) => setChecked(event.target.checked)}
    />
  );
};
```

## Indeterminate state

Use `isIndeterminate` to set the checkbox into an indeterminate state.

<Preview>
  <CheckboxField label="Include receipt" isIndeterminate />
</Preview>

```tsx sandbox
<CheckboxField label="Include receipt" isIndeterminate />
```

## Accessibility

CheckboxField adheres to the [checkbox role requirements](https://www.w3.org/TR/wai-aria-1.2/#checkbox)

### Keyboard Interaction

- `tab`: Moves focus to the element or outside
- `Space`: Toggles checkbox between checked and unchecked states
