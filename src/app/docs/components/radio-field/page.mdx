:::warning{title="The RadioField should only be used within a RadioGroup"}
The RadioField component should be wrapped inside a RadioGroup. Learn more
about the RadioGroup [here](/docs/components/radio-group).
:::

## Basic usage

<Preview>
  <DemoRadioField />
</Preview>

```tsx
const Demo = () => {
  const [isChecked, setIsChecked] = useState(false);

  return (
    <RadioField
      name="approval"
      value="approval-needed"
      label="Approval Needed"
      isChecked={isChecked}
      onChange={(e) => {
        setIsChecked(true);
      }}
    />
  );
};
```
