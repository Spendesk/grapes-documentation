## Use an icon

Use the `iconName` prop to set a icon on the left side of the Tag.

<Preview>
  <Tag variant="purple" iconName="plane">
    My tag
  </Tag>
</Preview>

```tsx sandbox
<Tag variant="purple" iconName="plane">
  My tag
</Tag>
```
