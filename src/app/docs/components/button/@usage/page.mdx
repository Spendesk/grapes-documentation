## Variant

The Button comes with three main visual styles: primary, secondary (default), and tertiary.

### Primary

Use the primary Button for the main action in a page.

<Preview>
  <Button variant="primaryBrand" text="Label" />
  <Button variant="primaryInfo" text="Label" />
  <Button variant="primarySuccess" text="Label" />
  <Button variant="primaryWarning" text="Label" />
  <Button variant="primaryAlert" text="Label" />
</Preview>

```tsx sandbox
<Preview>
```

## With icons

You can attach an Icon to the Button using `iconName` prop.

Use the `iconPosition` prop to set the position of the button, either left (default) or right

<Preview>
  <Button iconName="caret-left" text="Go back" />
  <Button iconName="caret-right" iconPosition="right" text="Continue" />
</Preview>

```tsx sandbox
<>
  <Button iconName="caret-left" text="Go back" />
  <Button iconName="caret-right" iconPosition="right" text="Continue" />
</>
```

## Loading state

There are two ways to set the Button's state as loading:

- The `isLoading` prop
- The asynchronous `onClick` function

<Preview>
  <Button isLoading variant="primaryBrand" text="Primary" />
  <Button isLoading variant="secondaryNeutral" text="Secondary" />
  <Button isLoading variant="tertiaryBrand" text="Tertiary" />
</Preview>

```tsx
<>
  <Button isLoading {...otherProps} />
  <Button
    onClick={async (event) => {
      await submitForm(event);
    }}
    {...otherProps}
  />
</>
```
