## Variant

The Button comes with six visual styles: primary (default), secondary, contrasted, warning, alert, and ghost.

<Preview>
  <Button variant="primary" text="Primary" />
  <Button variant="secondary" text="Secondary" />
  <Button variant="contrasted" text="Contrasted" />
  <Button variant="warning" text="Warning" />
  <Button variant="alert" text="Alert" />
  <Button variant="ghost" text="Ghost" />
</Preview>

```tsx
<>
  <Button variant="primary" text="Primary" />
  <Button variant="secondary" text="Secondary" />
  <Button variant="contrasted" text="Contrasted" />
  <Button variant="warning" text="Warning" />
  <Button variant="alert" text="Alert" />
  <Button variant="ghost" text="Ghost" />
</>
```

## Size

Use the `fit` props to control the size of the Button.

<Preview>
  <Button fit="content" text="Content fit" />
  <Button fit="parent" text="Parent fit" />
</Preview>

```tsx
<>
  <Button fit="content" text="Content fit" />
  <Button fit="parent" text="Parent fit" />
</>
```

## With icons

You can attach an Icon to the Button using `iconName` prop.

Use the `iconPosition` prop to set the position of the button, either left (default) or right

<Preview>
  <Button iconName="caret-left" text="Go back" />
  <Button iconName="caret-right" iconPosition="right" text="Continue" />
</Preview>

```tsx
<>
  <Button iconName="caret-left" text="Go back" />
  <Button iconName="caret-right" iconPosition="right" text="Continue" />
</>
```

## Text size

Use the `maxTextWidth` prop to set a maximum size for the text. An ellipsis will automatically be created if the text overflows the given size.

<Preview>
  <Button maxTextWidth="10ch" text="This is a very long text" />
</Preview>

```tsx
<>
  <Button maxTextWidth="10ch" text="This is a very long text" />
</>
```

## Loading state

There are two ways to set the Button's state as loading:

- The `isLoading` prop
- The asynchronous `onClick` function

<Preview>
  <Button isLoading variant="primary" text="Primary" />
  <Button isLoading variant="secondary" text="Secondary" />
  <Button isLoading variant="contrasted" text="Contrasted" />
  <Button isLoading variant="warning" text="Warning" />
  <Button isLoading variant="alert" text="Alert" />
  <Button isLoading variant="ghost" text="Ghost" />
</Preview>

```tsx
<>
  <Button isLoading {...otherProps} />
  <Button
    onClick={async (event) => {
      await submitForm(event);
    }}
    {...otherProps}
  />
</>
```
