import { Preview as GrapesPreview, Icon, Tag } from "@dev-spendesk/grapes";
import { Preview } from "@/components/preview/preview";

## Basic usage

The `rightAddon` prop is used to add a component on the right of the `Preview` component. To add a callback to the component, for example to open a modal or redirect to another page, you can use the `onClick` prop.

<Preview>
  <GrapesPreview
    primaryText="Uploaded on November 4, 2020"
    rightAddon={
      <Icon aria-hidden="true" color="var(--color-neutral)" name="search" />
    }
  />
</Preview>

```tsx
<Preview
  primaryText="Uploaded on November 4, 2020"
  rightAddon={
    <Icon aria-hidden="true" color="var(--color-neutral)" name="search" />
  }
  onClick={() => openModal}
/>
```

## Icon

The `iconName` prop is used to add a `HighlightIcon` component to the left of the `Preview` component.

<Preview>
  <GrapesPreview
    iconName="receipt-checked"
    primaryText="Uploaded on November 4, 2020"
    rightAddon={
      <Icon aria-hidden="true" color="var(--color-neutral)" name="search" />
    }
  />
</Preview>

```tsx
<Preview iconName="receipt-checked" {...otherProps} />
```

## Secondary text

The `secondaryText` prop allows you to specify a secondary text to display. You can use a string or a component as the secondary text.

<Preview>
  <GrapesPreview
    primaryText="€400.00"
    secondaryText="From your bank"
    rightAddon={
      <>
        <Tag variant="success" className="mr-xs">
          Pending
        </Tag>
        <Icon
          aria-hidden="true"
          color="var(--color-neutral)"
          name="caret-right"
        />
      </>
    }
  />
</Preview>

```tsx
<Preview primaryText="€400.00" secondaryText="From your bank" {...otherProps} />
```

## Overflowing secondary text

If the `fit` prop is set to `parent` and the content in the secondary text overflows, an ellipsis will appear.

<Preview>
  <div className="w-[360px]">
    <GrapesPreview
      fit="parent"
      primaryText="Main text"
      secondaryText="Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
    />
  </div>
</Preview>

```tsx
<div className="w-[360px]">
  <Preview
    fit="parent"
    secondaryText="Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
    {...otherProps}
  />
</div>
```

## Highlighted

The `isHighlighted` prop lets you specify whether the `Preview` component should be highlighted.

<Preview>
  <GrapesPreview
    isHighlighted
    iconName="receipt-checked"
    primaryText="Uploaded on November 4, 2020"
    rightAddon={
      <Icon aria-hidden="true" color="var(--color-neutral)" name="search" />
    }
  />
</Preview>

```tsx
<Preview isHighlighted {...otherProps} />
```
