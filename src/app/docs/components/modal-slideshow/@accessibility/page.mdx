## Summary

| Specification      | Value      |
| :----------------- | :--------- |
| Aria Role          | `modal`    |
| Accessibility name | `carousel` |

#### Carousel

- The carousel container has the `group` role.
- The carousel container has the aria-roledescription property set to `carousel`.

#### Slide

- Each slide container has the `group` role with the property aria-roledescription set to `slide`.
- Each slide has an accessible name given by the prop `title`
