## Basic usage

<Preview>

```tsx
const Demo = () => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button
        variant="primaryBrand"
        text="Open the modal"
        onClick={() => setIsOpen(true)}
      />
      <PageModal isOpen={isOpen} onClose={() => setIsOpen(false)}>
        <div>
          <p>This is a page modal.</p>
        </div>
      </PageModal>
    </>
  );
};
```
