## Variant

The IconButton comes with three main visual styles: primary, secondary (default), and tertiary.

### Primary

Use the primary IconButton for the main action in a page.

<Preview>
  <IconButton variant="primaryBrand" iconName="dots" />
  <IconButton variant="primaryInfo" iconName="dots" />
  <IconButton variant="primarySuccess" iconName="dots" />
  <IconButton variant="primaryWarning" iconName="dots" />
  <IconButton variant="primaryAlert" iconName="dots" />
</Preview>

## Margin on tertiary variant

Use the prop `hasNegativeMargins` to remove the margin around a tertiary IconButton.
This prop doesn't have any impact on the primary and secondary IconButton.

<Preview>
  <IconButton iconName="dots" variant="tertiaryNeutral" hasNegativeMargins />
</Preview>

```tsx sandbox
<IconButton iconName="dots" variant="tertiaryNeutral" hasNegativeMargins />
```

## Get element ref

Create a ref using `useRef` and pass it using the `ref` prop.

<Preview>
  <IconButton iconName="dots" variant="secondaryNeutral" />
</Preview>

```tsx sandbox
const Demo = () => {
  const ref = useRef<HTMLButtonElement | null>(null);

  return <IconButton ref={ref} iconName="dots" variant="secondaryNeutral" />;
};
```
