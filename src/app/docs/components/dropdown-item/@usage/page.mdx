## Description

The `helpText` prop is used to add a description to the DropdownItem.

<Preview>
  <DropdownItem label="Michael Murphy" helpText="Growth team" />
</Preview>

```tsx
<DropdownItem helpText="Growth team" {...otherProps} />
```

## Highlighted

The `isHighlighted` prop is used to highlight the DropdownItem.

<Preview>
  <DropdownItem label="Michael Murphy" isHighlighted />
</Preview>

```tsx
<DropdownItem isHighlighted {...otherProps} />
```

## Selected

The `isSelected` prop is used to display the DropdownItem as selected.

<Preview>
  <DropdownItem label="Michael Murphy" isSelected />
</Preview>

```tsx
<DropdownItem isSelected {...otherProps} />
```

## Prefix and suffix

Items can be added inside a DropdownItem with the `prefix` and `suffix` props.

<Preview>
  <DropdownItem
    label="Michael Murphy"
    prefix={
      <Avatar variant="square" size="s" src="/aurelien.webp" text="Aurelien" />
    }
  />
  <DropdownItem label="Michael Murphy" suffix={<span>24€</span>} />
</Preview>

```tsx
<>
  <DropdownItem
    prefix={
      <Avatar variant="square" size="s" src="/aurelien.webp" text="Aurelien" />
    }
    {...otherProps}
  />
  <DropdownItem suffix={<span>24€</span>} {...otherProps} />
</>
```
