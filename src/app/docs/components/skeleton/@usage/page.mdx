## Skeleton

:::warning{title="Usage"}
Predefined skeleton components are available, such as Avatar, Button, Table,
etc. You should only use this generic skeleton if none of the other components
fit your design.
:::

<Preview>
  <Skeleton height="24px" width="360px" />
</Preview>

```tsx sandbox
<Skeleton height="24px" width="360px" />
```

## SkeletonTable

This component provides a placeholder for the Table component. Like the Table component, some customization options are available:

- The `rowHeight` prop changes the size of the rows
- The `withHeader` prop displays the header if needed
- The `numberOfRows` prop allows to specify the number of rows to display (defaults to 5)

<Preview>
  <SkeletonTable
    numberOfRows={5}
    columns={[
      {
        cell: (
          <>
            <SkeletonCheckbox className="mr-s" />
            <SkeletonText width="80%" />
          </>
        ),
        header: (
          <>
            <SkeletonCheckbox className="mr-s" />
            <SkeletonText width="50%" />
          </>
        ),
      },
    ]}
    withHeader
  />
</Preview>

```tsx sandbox
<SkeletonTable
  numberOfRows={5}
  columns={[
    {
      cell: (
        <>
          <SkeletonCheckbox className="mr-s" />
          <SkeletonText width="80%" />
        </>
      ),
      header: (
        <>
          <SkeletonCheckbox className="mr-s" />
          <SkeletonText width="50%" />
        </>
      ),
    },
  ]}
  withHeader
/>
```
