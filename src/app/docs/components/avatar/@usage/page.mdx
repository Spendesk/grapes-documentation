## Fallback

If there is an error loading the src of the avatar, there are 2 fallbacks:

- If there's a `fallbackSrc` prop, we use it.
- If there's no `fallbackSrc` prop, we use the text props to generate the initials and a random color for both the background and the text color.

<Preview>
  <Avatar
    fallbackSrc="/supplier.svg"
    size="m"
    text="Spendesk"
    variant="square"
  />
  <Avatar size="s" text="George Gray" />
  <Avatar size="s" text="Lewis Barker" />
  <Avatar size="s" text="Michael Murphy" />
  <Avatar size="s" text="Nayden Lennart" />
</Preview>

```tsx
<>
  <Avatar
    fallbackSrc="/supplier.svg"
    size="m"
    text="Spendesk"
    variant="square"
  />
  <Avatar size="s" text="George Gray" />
  <Avatar size="s" text="Lewis Barker" />
  <Avatar size="s" text="Michael Murphy" />
  <Avatar size="s" text="Nayden Lennart" />
</>
```
