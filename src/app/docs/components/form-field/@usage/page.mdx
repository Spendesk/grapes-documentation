## Description

Use the `description` prop to add a description to the field.

<Preview>
  <Demo label="Your name" description="This is an optional description."/>
</Preview>
```tsx

  label="Your name"
  description="This is an optional description."
>
  <TextInput {...textInputProps} />
</FormField>
```

## Error message

Use the `alertMessage` props to display an alert message below the field.
When given, the FormField will also make any input children invalid and set the right html and aria attributs to it.

<Preview>
  <Demo label="Your name" alertMessage="This information is required."/>
</Preview>
```tsx
<FormField label="Your name" alertMessage="This information is required.">
  <TextInput {...textInputProps} />
</FormField>
```

## Best practices

In situations where you need to use a FormField without a label, we recommend that you still give a label but add the prop `visuallyHideLabel`.
Doing so, your input will be labelled by your label, making your app more accessible and easier to test. 

```tsx
<FormField 
  label="Your name"
  visuallyHideLabel // [!code highlight]
>
  <TextInput {...textInputProps} />
</FormField>
```
