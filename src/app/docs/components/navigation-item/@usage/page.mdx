## Variant

The navigation item comes with three visual styles: `mainNavigation`, `subNavigation`, and `sideNavigation`.

<Preview className="items-center">
  <div className="bg-complementary h-[68px] flex items-center justify-center px-s">
    <NavigationItem text="Main navigation item" variant="mainNavigation" />
  </div>
  <NavigationItem text="Sub navigation item" variant="subNavigation" />
  <NavigationItem text="Side navigation item" variant="sideNavigation" />
</Preview>

```tsx sandbox
<>
  <NavigationItem text="Main navigation item" variant="mainNavigation" />
  <NavigationItem text="Sub navigation item" variant="subNavigation" />
  <NavigationItem text="Side navigation item" variant="sideNavigation" />
</>
```

## Addon

The `leftAddon` and `rightAddon` props are used to display another element inside the navigation item.

<Preview>
  <NavigationItem
    text="Navigation item"
    variant="sideNavigation"
    leftAddon={<Badge>12</Badge>}
  />
  <NavigationItem
    text="Navigation item"
    variant="sideNavigation"
    rightAddon={<Badge>12</Badge>}
  />
</Preview>

```tsx
<>
  <NavigationItem leftAddon={<Badge>12</Badge>} {...otherProps} />
  <NavigationItem rightAddon={<Badge>12</Badge>} {...otherProps} />
</>
```

## Active

The `isActive` prop is used to display the navigation item as active.

<Preview className="items-center">
  <div className="bg-complementary h-[68px] flex items-center justify-center px-s">
    <NavigationItem
      isActive
      text="Main navigation item"
      variant="mainNavigation"
    />
  </div>
  <NavigationItem isActive text="Sub navigation item" variant="subNavigation" />
  <NavigationItem
    isActive
    text="Side navigation item"
    variant="sideNavigation"
  />
</Preview>

```tsx
<>
  <NavigationItem isActive variant="mainNavigation" {...otherProps} />
  <NavigationItem isActive variant="subNavigation" {...otherProps} />
  <NavigationItem isActive variant="sideNavigation" {...otherProps} />
</>
```

  <NavigationItem isEmphasized variant="subNavigation" {...otherProps} />
  <NavigationItem isEmphasized variant="sideNavigation" {...otherProps} />

## Divider

The `withDivider` prop is used to add a divider on the right of the navigation item. It acts as a separator between two items.

<Preview className="items-center">
  <div className="bg-complementary h-[68px] flex items-center justify-center px-s">
    <NavigationItem
      withDivider
      text="Main navigation item"
      variant="mainNavigation"
    />
  </div>
  <NavigationItem
    withDivider
    text="Sub navigation item"
    variant="subNavigation"
  />
  <NavigationItem
    withDivider
    text="Side navigation item"
    variant="sideNavigation"
  />
</Preview>

```tsx
<>
  <NavigationItem withDivider variant="mainNavigation" {...otherProps} />
  <NavigationItem withDivider variant="subNavigation" {...otherProps} />
  <NavigationItem withDivider variant="sideNavigation" {...otherProps} />
</>
```
