# Supported platforms

## Browser

Grapes supports the latest, stable releases of all major browsers.

| Browser          | Version |
| :--------------- | :------ |
| `Edge`           | >= 99   |
| `Chrome`         | >= 99   |
| `Safari (macOS)` | >= 16.5 |
| `Firefox`        | >= 115  |

## React

Grapes supports the most recent versions of React, starting with **18.2.0**.
