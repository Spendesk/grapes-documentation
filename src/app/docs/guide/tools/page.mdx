# Tools

## Grapes for VS Code

VS Code extension for effortless CSS writing with Grapes, providing autocomplete suggestions for CSS properties.

:::info{title="Requirement"}
Grapes for VS Code requires Grapes 1.x or above.
:::

### How to use it

1. Install the [Grapes VS Code extension](https://marketplace.visualstudio.com/items?itemName=Spendesk.grapes-vscode)
2. Open a CSS file and start typing. The extension is triggered with the characters: `--`
