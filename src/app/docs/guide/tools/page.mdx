# Tools

## Grapes for VS Code

VS Code extension for effortless CSS writing with Grapes, providing autocomplete suggestions for CSS properties.

:::info{title="Requirement"}
Grapes for VS Code requires Grapes 1.x or above.
:::

### How to use it

1. Navigate to [the github repository](https://github.com/Spendesk/grapes-vscode/releases)
2. Download the file ` grapes-vscode-X.X.X.vsix` and install it
3. Open a CSS file and start typing. The extension is trigger with the characters: `--`
