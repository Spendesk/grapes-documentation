# Supported platforms

## Browser

Grapes supports the latest, stable releases of all major browsers.

| Browser          | Version |
| :--------------- | :------ |
| `Edge`           | >= 105  |
| `Chrome`         | >= 105  |
| `Safari (macOS)` | >= 16.5 |
| `Firefox`        | >= 121  |

## React

Grapes supports the most recent versions of React, including React **18.2.0** and React **19**.
